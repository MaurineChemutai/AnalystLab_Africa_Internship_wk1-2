# 📊 Data Cleaning & Exploratory Data Analysis (EDA)

> A comprehensive data preprocessing and exploratory data analysis project using Python to transform raw data into meaningful insights.

---

## 📖 Project Overview

Data quality is the foundation of every successful data science and machine learning project. Before building predictive models, datasets must be cleaned, validated, and thoroughly explored.

This project demonstrates a complete data cleaning and exploratory data analysis (EDA) workflow on multiple real-world datasets. It covers identifying missing values, detecting duplicates, understanding feature distributions, and visualizing relationships between variables to uncover valuable insights.

The notebook follows best practices used in professional data analytics and machine learning projects.

---

## 🎯 Objectives

- Import and inspect datasets
- Understand dataset structure
- Detect and handle missing values
- Check for duplicate records
- Explore numerical and categorical features
- Perform statistical analysis
- Visualize data distributions
- Identify relationships between variables
- Prepare clean datasets for machine learning

---

## 📂 Datasets Used

### 🚢 Titanic Dataset
The Titanic dataset is used to analyze passenger information and explore factors that influenced survival.

Key features include:

- Passenger information
- Age
- Gender
- Passenger class
- Fare
- Embarkation point
- Survival status

---

### 🏠 Housing Dataset

The Housing dataset is used to analyze different housing characteristics and understand how various factors relate to house prices.

Features include:

- Area
- Bedrooms
- Bathrooms
- Stories
- Parking
- Furnishing status
- House price
